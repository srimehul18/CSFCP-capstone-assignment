# System Uptime Status Script

A simple Bash script that displays key system information such as uptime, date/time, logged-in user, RAM usage, and disk space.

---

##  Features
- Shows **system uptime**
- Displays **current date and time**
- Prints **logged-in user**
- Shows **memory usage**
- Displays **root disk space**
